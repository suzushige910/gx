# GX Smart Life WordPress Theme

このリポジトリには、GXスマートライフ株式会社のWordPressテーマが含まれています。

## 概要

千葉県市原市を拠点とする再生可能エネルギー会社「GXスマートライフ」の公式WordPressテーマです。
太陽光発電・蓄電池システムの販売・施工サービスを紹介するWebサイト用に設計されています。

## 主な機能

- レスポンシブデザイン（モバイル・タブレット・デスクトップ対応）
- Swiperカルーセル（トップページ）
- カスタム投稿タイプ（施工例）
- 会社プロフィールウィジェット
- LINE公式アカウント連携
- Google Analytics 4対応
- SEO最適化（Schema.org対応）

## 自動デプロイ

mainブランチへのpushで自動的に本番サーバーにデプロイされます。

## ディレクトリ構成

- `gx-smartlife-theme/` - WordPressテーマ本体
- `.github/workflows/` - GitHub Actions デプロイメント設定
