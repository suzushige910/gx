# GX Smart Life Theme インストールガイド

このドキュメントでは、GX Smart Life WordPressテーマのインストールと初期設定方法を説明します。

## 必要要件

- WordPress 5.8以上
- PHP 7.4以上
- MySQL 5.7以上 または MariaDB 10.3以上

## インストール手順

### 1. テーマファイルのアップロード

#### 方法A: WordPress管理画面から
1. WordPress管理画面にログイン
2. 「外観」→「テーマ」に移動
3. 「新規追加」をクリック
4. 「テーマのアップロード」をクリック
5. `gx-smartlife-theme.zip`を選択してアップロード
6. 「今すぐインストール」をクリック
7. インストール完了後「有効化」をクリック

#### 方法B: FTPでアップロード
1. FTPクライアント（FileZilla等）でサーバーに接続
2. `/wp-content/themes/`ディレクトリに移動
3. `gx-smartlife-theme`フォルダをアップロード
4. WordPress管理画面「外観」→「テーマ」から有効化

### 2. 推奨プラグインのインストール

以下のプラグインをインストールして有効化してください：

#### 必須プラグイン
- **Contact Form 7** - お問い合わせフォーム
  - WordPress管理画面「プラグイン」→「新規追加」で検索してインストール

#### 推奨プラグイン
- **Yoast SEO** または **Rank Math** - SEO最適化
- **Autoptimize** - CSS/JavaScriptの最適化
- **EWWW Image Optimizer** - 画像の最適化
- **WP Super Cache** または **W3 Total Cache** - キャッシュ管理

### 5. Contact Form 7の設定

1. **WordPress管理画面「お問い合わせ」→「新規追加」**
2. フォーム名: 「お問い合わせ」
3. 以下のフォームコードを入力:

```html
<label> お名前（必須）
    [text* your-name] </label>

<label> メールアドレス（必須）
    [email* your-email] </label>

<label> 電話番号
    [tel your-tel] </label>

<label> お問い合わせ種別（必須）
    [select* your-subject "卒FIT対策" "BCP対策・UPS" "太陽光発電施工" "パートナー募集" "その他"] </label>

<label> お問い合わせ内容（必須）
    [textarea* your-message] </label>

<label>
    [acceptance acceptance-privacy] <a href="/privacy-policy" target="_blank">プライバシーポリシー</a>に同意する（必須）
</label>

[submit "送信する"]
```

4. 「保存」をクリック
5. ショートコード `[contact-form-7 id="1" title="お問い合わせ"]` をコピー
6. お問い合わせページに貼り付け

## トラブルシューティング

### Q: カルーセルが表示されない
A:
- ブラウザのキャッシュをクリア
- Swiperライブラリが正しく読み込まれているか確認
- ブラウザのコンソールでJavaScriptエラーを確認

### Q: メニューが表示されない
A:
- メニューが作成され、メニューの位置が設定されているか確認
- フォールバックメニューが表示されているか確認

### Q: レイアウトが崩れる
A:
- ブラウザキャッシュをクリア
- CSSファイルが正しく読み込まれているか確認
- プラグインの競合を確認（一時的にすべて無効化してテスト）

### Q: 画像が表示されない
A:
- 画像ファイルが正しくアップロードされているか確認
- ファイルパーミッションを確認（644推奨）
- メディアライブラリで画像を確認

## パフォーマンス最適化

### 推奨設定

1. **画像の最適化**
   - EWWW Image Optimizer プラグインで自動最適化
   - WebP形式への変換を有効化

2. **キャッシュの有効化**
   - WP Super Cache または W3 Total Cache をインストール
   - ブラウザキャッシュを有効化

3. **CSS/JavaScriptの最適化**
   - Autoptimize プラグインで圧縮・結合
   - 遅延読み込みを有効化

4. **CDNの利用**（オプション）
   - Cloudflare などのCDNサービスを設定

## セキュリティ対策

1. **定期的な更新**
   - WordPress本体、テーマ、プラグインを常に最新版に

2. **セキュリティプラグイン**
   - Wordfence または iThemes Security をインストール

3. **バックアップ**
   - UpdraftPlus などでバックアップを自動化

4. **SSL証明書**
   - Let's Encrypt などでSSL/TLS暗号化を有効化
